Configuration

  1. Go to OpenRouter AI > Configurations
  2. Enter your OpenRouter API key (get one free at openrouter.ai/keys)
  3. Set your App URL and App Name (recommended for OpenRouter ranking)
  4. Click Sync Models to fetch the latest 700+ model catalog
  5. Browse models — free models show a 🆓 badge
  6. Click Open Chat on any model to start using AI
