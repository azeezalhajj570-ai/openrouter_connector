Seamlessly integrate OpenRouter's AI model catalog with Odoo.

Features:
- Adds OpenRouter as an AI provider in Odoo's AI framework
- Syncs the complete OpenRouter model catalog (700+ models)
- Chat completion with usage tracking and cost logging
- Token and cost reconciliation via OpenRouter generation API
- Tool calling and multi-modal model support
- Automatic tracking of prompt, completion, reasoning, cached, and audio tokens
- Full request/response payload logging with attachment offloading for large payloads
